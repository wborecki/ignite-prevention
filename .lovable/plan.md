

# WI-Fire - Engenharia Contra Incêndio

Site institucional clean e profissional, focado em conversão (contato via WhatsApp) e SEO.

## Cores e Design
- **Cor primária**: Vermelho escuro/bordô da marca (#7A0000)
- **Secundária**: Branco e cinza claro para fundo limpo
- **Tipografia**: Sans-serif moderna, hierarquia clara
- **Estilo**: Clean, profissional, sem animações desnecessárias, sem badges

## Páginas e Seções

### 1. Home
- **Hero**: Título forte + subtítulo sobre engenharia contra incêndio + CTA direto para WhatsApp
- **Serviços**: Grid com os 6 serviços (PPCI, RPCI, AVCB, Habite-se, Alvarás/Laudos, Inspeções) — ícones minimalistas, texto objetivo
- **Projetos Realizados**: Galeria com fotos reais de equipamentos instalados (extintores, sprinklers, hidrantes, sinalização, etc.) — layout em grid limpo
- **CTA intermediário**: Faixa com chamada para contato via WhatsApp
- **Sobre resumido**: Breve texto institucional com link para página Sobre

### 2. Soluções
- Página dedicada com detalhamento de cada serviço
- Descrição clara do que é cada serviço, para quem serve

### 3. Sobre
- História da empresa, missão, valores
- Foto/equipe (placeholder profissional por enquanto)

### 4. Blog
- Listagem de posts com imagem, título, resumo e data
- Página individual de post
- 3-4 artigos pré-escritos com conteúdo relevante para SEO (ex: "O que é PPCI?", "Como obter o AVCB", "Importância da inspeção de extintores", "Diferença entre PPCI e RPCI")
- Imagens profissionais reais (fotos de stock de equipamentos contra incêndio)

### 5. Contato
- Formulário simples (nome, email, telefone, mensagem)
- Informações de contato (telefone, email, endereço)
- Mapa placeholder

## Elementos Globais
- **Navbar**: Logo + Home | Soluções | Sobre | Blog | Contato — limpa, sticky
- **Botão WhatsApp flutuante**: Fixo no canto inferior direito, sempre visível
- **Footer**: Links das páginas, redes sociais (Instagram, Facebook, LinkedIn), informações de contato, copyright
- **SEO**: Meta tags, títulos H1/H2 otimizados, textos com palavras-chave relevantes, estrutura semântica

## Projetos/Portfólio (seção na Home)
- Grid de fotos profissionais de equipamentos e instalações (extintores, sprinklers, hidrantes, painéis de alarme, sinalização)
- Usando imagens de stock profissionais via Unsplash
- Legendas descritivas
