# WI-FIRE Engenharia Contra Incêndio

Site institucional da WI-FIRE — empresa de engenharia contra incêndio em Itajaí, SC.

## Tecnologias

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router DOM
- TanStack React Query

## Desenvolvimento

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```
